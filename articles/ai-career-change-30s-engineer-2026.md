## ステップ1：自己分析をAIで完全言語化

30代の転職で最初の壁は「自分の価値を言葉にすること」です。

### プロンプト：経験の棚卸し

```
あなたはシニアエンジニアのキャリアコンサルタントです。
私の経験を整理して、市場価値の高い強みとして言語化してください。

【経験の箇条書き】
- {経験1（例：マイクロサービス設計・導入）}
- {経験2（例：チーム5名のリード）}
- {経験3（例：レガシーシステムのリファクタリング）}
- {経験4}
- {経験5}

以下の形式で整理してください：
1. 「技術的強み」トップ3（具体的な成果数値付き）
2. 「マネジメント・リーダーシップ強み」トップ2
3. 30代転職市場での「差別化ポイント」
4. どんな企業・ポジションに最もフィットするか
```

### 実行例

**入力した経験**:
- ECサイトのバックエンドをモノリスからマイクロサービスに分割
- 8名チームのテックリード
- レスポンスタイムを平均3秒→0.3秒に改善

**AIが言語化した強み**:
> 「大規模システムのアーキテクチャ改善（マイクロサービス化）を主導し、パフォーマンスを10倍改善した実績を持つシニアバックエンドエンジニア。8名チームのテックリードとして技術選定・コードレビュー・メンタリングを担当。」

## ステップ2：ターゲット企業選定をAIで自動化

「どの企業を受けるべきか」も30代の悩みの一つ。AIで絞り込みを効率化します。

### プロンプト：企業マッチング分析

```
以下の私のプロフィールに最も合う企業タイプを分析してください。

【私のプロフィール】
- 年齢: 35歳
- 現職: バックエンドエンジニア（8年）
- 技術スタック: {Go, Python, AWS, PostgreSQL など}
- 希望年収: ¥{900万}〜¥{1,200万}
- 働き方: {フルリモート希望}
- やりたいこと: {プロダクト開発に近い環境でアーキテクチャ設計をしたい}
- 避けたいこと: {SES・客先常駐}

【分析してほしいこと】
1. 私に最もフィットする企業タイプ（スタートアップ/メガベンチャー/事業会社/etc）
2. 狙うべき職種・ポジション名（求人検索キーワード）
3. 避けるべき企業の特徴
4. 転職成功確率が高い業界Top3とその理由
```

## ステップ3：30代向け職務経歴書の最適化

30代の職務経歴書は「技術 × リーダーシップ × ビジネス貢献」の三角形で構成します。

### プロンプト：職務経歴書 強化版

```
以下の職務経歴書を、30代シニアエンジニアとして最大限魅力的に書き直してください。

【元の職務経歴】
{現在の職務経歴書の内容}

【ターゲット求人のキーワード】
{求人票から抽出した重要キーワード}

【強調してほしい点】
- 技術的判断・アーキテクチャ設計の経験
- チームへの影響・成果
- ビジネス貢献（売上・コスト削減・品質向上の数値）

【フォーマット要件】
- 各プロジェクトはSTAR形式（状況・課題・行動・成果）
- 数値・具体例を必ず入れる
- ATS対策としてキーワードを自然に盛り込む
- 1ページ目で採用担当者を引きつける要約を冒頭に
```
