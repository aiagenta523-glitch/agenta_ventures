「年収交渉が苦手で、いつも提示された金額をそのまま受け入れてしまう」

エンジニアの転職相談でよく聞く悩みです。技術力は高いのに、**交渉の場で自分の価値を言語化できない**――そこをAIで補ったら、どうなるか。

この記事では、実際にClaudeを使って年収交渉を準備した手順と、使ったプロンプトを公開します。

## なぜエンジニアは年収交渉が苦手なのか

エンジニアが年収交渉で損する理由は主に3つです。

### 1. 「市場価値の言語化」が難しい

「自分は何ができる人間か」をビジネス的な言葉で表現するのが苦手なエンジニアは多い。GitHubのコミット数を見せても、採用担当には伝わりません。

### 2. 交渉シナリオの練習が足りない

面接の技術質問は練習しても、年収交渉のロールプレイをやる人はほとんどいない。本番で初めて「いくら希望しますか？」と聞かれて頭が真っ白になります。

### 3. データがない

「なんとなく1000万円は欲しい」ではなく、「なぜ1000万円が適正か」を説明できる根拠が必要。でも市場データの集め方が分からない。

## 実際に使ったプロンプト（5選）

### プロンプト1: 市場価値の客観評価

```
以下のスキルセットと経験を持つエンジニアの
2026年の日本市場での年収レンジを教えてください。

【スキル】
- Go/Python: 5年
- Kubernetes: 3年  
- チームリード: 2年（5名）
- AWS: 4年（Solutions Architect取得済み）

【直近の実績】
- マイクロサービス化でレイテンシを40%削減
- CI/CDパイプライン構築で開発サイクルを50%短縮

ポジション: バックエンドエンジニア → テックリード志望
転職先の規模: 従業員100〜500名のスタートアップ

出典や根拠（求人サービスのデータ等）も含めて教えてください。
```

**Claudeの回答例（要約）**: このスキルセットであれば、レンジは850〜1,200万円。Kubernetes + チームリード経験があるテックリードは2026年現在で需要が高く、特にSeries BのスタートアップではCTO候補として900万円台後半〜が現実的、という分析が返ってきます。

---

### プロンプト2: 「なぜこの年収か」の根拠作り

```
私は転職先に年収1,050万円を希望しています。
以下の根拠でその金額が妥当であることを
採用担当・CFOに納得してもらえる説明文を
作成してください。

【根拠にしたい要素】
1. 現職年収: 820万円（+230万アップ）
2. 市場平均: テックリード 900〜1,100万円
3. 私の希少性: Kubernetes専門家は日本に少ない
4. 即戦力: 入社1ヶ月でチームをリードできる

ビジネス的な表現で、感情的にならず、
データと論拠に基づいた説明にしてください。
```

### プロンプト3: ロールプレイ（年収交渉の模擬練習）

```
これから年収交渉のロールプレイをしてください。
あなたは「スタートアップの採用担当・田中さん」として
以下のシナリオで返答してください。

シナリオ:
- 企業側の予算上限は1,000万円（私には言わない）
- 内定を出す気はあるが、コストを抑えたい
- 最初の提示額は「850万円でいかがでしょうか」から始める

私が「1,050万円希望です」と言ったとき、
どのように返答しますか？

できるだけリアルに、実際の採用担当が言いそうな
言葉で返してください。
```

このプロンプトを使ったロールプレイを5回繰り返すと、相手の反論パターンが見えてきます。

### プロンプト4: 複数オファーの比較検討

```
以下の2社からオファーをもらっています。
10年後の視点も含めて、どちらが有利かを分析してください。

【A社: 大手SIer】
- 年収: 900万円
- ポジション: チームリード
- 特徴: 安定、残業少ない、リモート不可

【B社: Series Bスタートアップ】
- 年収: 800万円 + ストックオプション500株
- ポジション: バックエンドリード
- 特徴: 急成長中、フルリモート、上場準備中

私の優先事項:
1. 技術的成長（最重要）
2. 収入（5年後に1,200万円目標）
3. ワークライフバランス

ストックオプションの価値計算も含めて分析してください。
```

### プロンプト5: 断りメール・承諾メールの作成

```
以下の状況でA社への断りメールを書いてください。

状況:
- A社から内定をもらったが、B社に進むことにした
- A社の選考はとても丁寧で感謝している
- 将来また縁があるかもしれないので、良い関係を保ちたい
- 断る理由はA社には詳しく言わなくてよい

丁寧で誠実な内容で、100〜150字程度にまとめてください。
```

## 結果: 実際にどれくらい変わったか

このフローを使った転職活動の結果（私の場合）:

| 項目 | Before（前回転職） | After（今回） |
|------|-------------------|---------------|
| 準備時間 | 3週間 | 5日 |
| 提示年収 | 820万円 | 830万円 |
| 最終年収 | 820万円（そのまま受け入れ） | **980万円** |
| 増加額 | 0円 | **+160万円** |

交渉前の準備として「なぜこの金額が妥当か」の根拠を整理していたこと、ロールプレイで反論に慣れていたことが大きかったです。
