## はじめに

「転職相談にAIを使いたいけど、ChatGPTとClaude、どっちがいいの？」

この疑問を持つエンジニアは多いはずです。私は両方を実際に転職活動で使い込みました。この記事では忖度なしで比較します。

---

## 結論（先に言います）

| 用途 | 推奨 |
|------|------|
| 職務経歴書の添削 | **Claude** |
| 面接対策・ロールプレイ | **Claude** |
| 求人票の分析・比較 | **ChatGPT** |
| 給与交渉スクリプト | **Claude** |
| 企業リサーチ・最新情報 | **ChatGPT**（Webあり） |
| 長文の書類一括処理 | **Claude** |

**総合**: 転職相談の核心（書類・面接）は**Claudeが上**。情報収集は**ChatGPTが上**。

---

## 1. 職務経歴書の添削

### ChatGPTで試した場合

プロンプト：
```
この職務経歴書を添削してください。
[3000文字の職務経歴書を貼り付け]
```

**結果**: 文章は整うが、「具体的な数字を入れましょう」など一般的なアドバイスが多い。何度かやりとりしないと的を射た指摘が来ない。

### Claudeで試した場合

同じプロンプトを投げると、最初から：
- 「○○という表現は採用担当者に刺さりにくい。△△に書き換えると、XYZ株式会社の求人要件に直結します」
- 「この実績、数値化できますか？例えば"〜により開発速度が2倍"のように」

**具体的な改善案がセットで出てくる**のが特徴。「直してください」と言えばそのまま書き直してくれる。

### 軍配：Claude ⭐⭐⭐⭐⭐ vs ChatGPT ⭐⭐⭐

---

## 2. 面接ロールプレイ

面接対策でロールプレイをお願いした場合の比較です。

### ChatGPTの場合

面接官役をやってもらうと、質問は出てくるが**フィードバックが表面的**。「良い回答ですね。もう少し具体的に」程度。

### Claudeの場合

```
あなたはXYZ株式会社（SaaS企業）の技術面接官です。
私はバックエンドエンジニア（Go/5年）として応募しています。
一次面接を始めてください。終わったら評価をしてください。
```

このプロンプトで動かすと：

- 想定外の質問（「なぜGoを選んだ？Rustは検討しなかったの？」）
- 回答後の掘り下げ（「それって本番での話？テストの話？」）
- 面接後の詳細評価（強み・弱み・改善点を箇条書きで）

**本物の面接に近い緊張感と実践的フィードバック**がある。

### 軍配：Claude ⭐⭐⭐⭐⭐ vs ChatGPT ⭐⭐⭐

---

## 3. 企業リサーチ・最新情報収集

ここは**ChatGPTが圧勝**です。

ChatGPTはWebブラウジング機能（Plus）があるため：
- 最新の企業ニュース
- 採用ページのリアルタイム情報
- 直近の資金調達・事業展開

Claudeは学習データのカットオフがあり、最新情報は苦手（ただし公式発表では2025年以降も改善中）。

企業の現在地を把握するなら**ChatGPT + Web検索**が有効。

### 軍配：ChatGPT ⭐⭐⭐⭐⭐ vs Claude ⭐⭐⭐

---

## 4. 給与交渉スクリプト

実際に使ったプロンプトを公開します。

```
私はSaaS企業にバックエンドエンジニアとして内定をもらいました。
提示年収：600万円
希望年収：720万円
理由：現職が650万円で、転職するなら10%以上アップが条件

年収交渉のメールを書いてください。
失礼にならず、しかし明確に希望を伝えるトーンで。
```

**ChatGPTの出力**：丁寧だが若干テンプレ感。「ご検討いただけますと幸いです」が多用される。

**Claudeの出力**：
- 感謝と期待感を最初に示す
- 市場相場データへの言及（「同職種の中央値は〜」）
- 具体的な代替案（「一時金での補填でも〜」）
- 返信を促す自然な締め

実際にこのClaudeのスクリプトを使って、**600万→680万に交渉成功**しました（友人談）。

### 軍配：Claude ⭐⭐⭐⭐⭐ vs ChatGPT ⭐⭐⭐⭐

## 両方使う「ハイブリッド戦略」が最強

正直、どちらか一方に絞る必要はありません。私が推奨するフローは：

```
1. 企業リサーチ → ChatGPT（Web検索あり）
2. 求人票分析・マッチング度確認 → ChatGPT
         ↓
3. 職務経歴書作成・添削 → Claude
4. 面接ロールプレイ → Claude
5. 給与交渉スクリプト → Claude
```

**情報収集フェーズはChatGPT、書類・対話フェーズはClaude。**
